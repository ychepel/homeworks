1. **Задача: Обобщенный интерфейс для сравнения объектов**

   **Условие:**
   Определите обобщенный интерфейс `Comparer`, который имеет метод `compare`, сравнивающий два объекта.

   **Решение:**
   ```java
   public interface Comparer<T> {
       int compare(T obj1, T obj2);
   }
   ```
